# All Vap's Backend Root Only

Backend API pour All Vap's, préparé sans dossier pour un upload GitHub simple.

## Installation locale

```bash
npm install
npm run dev
```

## Render

Build Command :

```bash
npm install && npm run build
```

Start Command :

```bash
npm start
```

## Variables Render

```env
DATABASE_URL=
PORT=10000
NODE_ENV=production
FRONTEND_URL=
VIVA_CLIENT_ID=
VIVA_CLIENT_SECRET=
VIVA_MERCHANT_ID=
VIVA_API_URL=
JWT_SECRET=
```
